# Heart Disease Prediction Project
## Project Overview
Predicting Heart Disease using Machine 
Learning algorithms on 1000 patients data.

## Dataset
- Source: Kaggle
- Rows: 1000
- Columns: 16

## Steps Performed
1. Data Loading & Exploration
2. Missing Values Handling
3. Data Visualization (EDA)
4. Correlation Heatmap Analysis
5. Simple Linear Regression
6. Multiple Linear Regression

## Results
 Model      |   R2 Score
---------       --------
Simple LR   |     0.43 |
Multiple LR |     0.57 |

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Key Findings
- Age strongest predictor (0.65 correlation)
- Cholesterol second strongest (0.37)
- Multiple LR 14% better than Simple LR
